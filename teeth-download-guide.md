## Source
University of Dundee, School of Dentistry (@DundeeDental)
License: CC Attribution — https://creativecommons.org/licenses/by/4.0/

## Instructions

1. Go to each link below
2. Click the blue "Download 3D Model" button
3. Choose GLB format
4. Save the file to this folder: `public/teeth/`
5. Rename each file exactly as shown below

